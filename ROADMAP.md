# Day-by-Day Roadmap (Example 10 Days)

Day 1: Project setup, repo structure, CI pipeline, linting rules.
Day 2: Database schema, migrations, basic NestJS modules.
Day 3: Auth (register/login), JWT refresh flow, user profile.
Day 4: Topic search endpoints, YouTube API integration.
Day 5: Course list and details endpoints, caching layer.
Day 6: Flutter base app, routing, auth screens.
Day 7: Dashboard UI, search UI, analytics widgets.
Day 8: My Courses UI, course detail with video list.
Day 9: Profile UI, settings, saved/assessed flows.
Day 10: QA, test coverage, bug fixes, release checklist.
